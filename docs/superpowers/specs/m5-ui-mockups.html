Every cell is editable inline (data as text, options as dropdowns of the template's allowed values, so row 3's invalid CSV value is fixable here). Options come from the CSV or the manual strip above: an option.<name> column sets it per row (orientation), options the CSV omits use the manual control for all rows (outline), CSV wins when both exist. ⧉ duplicates a row, ✕ removes one, ↺ Reset restores removed rows and sets copies to 1. Copies multiplies every row uniformly: total = rows × copies (3 × 2 = 6, ordered 1,1,2,2,3,3), client-side. Download stays atomic; print reports {total, succeeded, failed[]}.
